# meowthon

这是我制作的解释型编程语言，名为meowthon，文件后缀名为me，使用python3编写。

运行：
首先安装python3，推荐使用最新版本
如果您使用python命令会运行python3，请使用以下代码：
```sh
python main.py
```
如果您使用python命令会运行python2，请使用以下代码：
```sh
python3 main.py
```
